# AI Based File Assistant

# Overview

This project is an AI-powered chatbot that analyzes text files and answers questions using LLM and RAG.

# Features

* Upload multiple text files
* Ask questions using AI
* Chat-based interface
* Source-based answers

# Tech Stack

* LangChain
* FAISS
* Groq LLM
* Streamlit

# How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
